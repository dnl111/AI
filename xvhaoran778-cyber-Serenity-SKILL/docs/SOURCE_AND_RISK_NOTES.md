# 来源与风险说明

## 来源

本 skill 来自对以下材料的抽象和整理：

- Serenity（`@aleabitoreddit`）公开推文和交易相关披露。
- 本地整理的 Serenity 推文、交易记录和社区解读。
- Serenity 风格案例中反复出现的研究模式：CPO、光通信、InP 衬底、硅光、先进封装、数据中心电力、RPI 小市值需求冲击等。

项目不包含大批原始推文或截图，只保留抽象后的研究流程、评分表和模式库。

## 与 Serenity 的关系

本项目与 Serenity 本人无关，不代表其观点，也不验证其身份、持仓或收益率。

Skill 的目标是复用研究方法，而不是复刻个人风格、传播收益神话或替代独立判断。

## 投资风险

- 小盘股和跨境股票可能流动性差、波动大、容易受融资和情绪影响。
- 供应链推断可能错误，客户关系和 BOM 可能保密或被误读。
- 政府补贴、客户验证、技术路线和行业标准都可能变化。
- 社交媒体披露不是审计后的交易记录。
- 任何输出都应视为研究辅助，不构成投资建议。

## 使用原则

1. 优先使用官方公告、交易所披露、公司 IR、财报和一手技术资料。
2. 新闻和社交媒体只能作为线索，不能单独作为结论。
3. 每个投资假设都必须写出反证和失效条件。
4. 对 A 股、美股、港股等不同市场使用不同信息源和风险检查清单。
5. 把“看好”和“明确买入/持有/卖出披露”严格分开。
